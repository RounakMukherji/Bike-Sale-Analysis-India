# Project Overview
This project focuses on analyzing bike sales data in India to uncover key business insights related to revenue, customer behavior, and product performance. Using SQL for data analysis and Power BI for visualization, the project answers real-world business questions that are commonly expected from a Data Analyst or Business Analyst role.

The dataset contains 10,000+ records across multiple tables and simulates a real sales environment with customers, bikes, and transactions.

# Tools & Technologies Used
- SQL (MySQL)
  - Joins
  - Subqueries
  - Window Functions
  - Aggregations
- Power BI
  - Data Modeling
  - DAX Measures
  - Interactive Dashboards
- Microsoft Excel

# Business Questions Answered
Some key business problems solved in this project:
- Top 5 bikes by total revenue
- Revenue contribution by bike type
- Monthly sales trend analysis
- Customer distribution by state and city tier
- Impact of payment mode (Cash / EMI / UPI) on sales
- Customer satisfaction analysis
- Sales performance using window functions

# Power BI Dashboard
The Power BI dashboard provides:
- Revenue and sales KPIs
- Top-performing bike models
- State-wise customer distribution
- Monthly sales trends
- Bike-type-wise revenue share

# Key Insights
- Certain bike models generate higher revenue due to repeated sales rather than high price.
- Metro and Tier-1 cities contribute the majority of total revenue.
- EMI-based purchases show higher average ticket size.
- Bike type plays a crucial role in revenue distribution.
